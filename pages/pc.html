princetoncourses

Responsive web app for intuitive browsing of course listings and evaluations. Serves over 5000 Princeton students each year. See iterative designs below. Created as a final project for COS 333: Advanced Programming Techniques.

Served as project manager and a member of front-end team for Princeton Courses. Worked with Sebastian Hallum Clarke, Caterina Golner, Mel Shu, and Bensu Sicim. Advised by Christopher Moretti. Implemented using NodeJS and MongoDB.
